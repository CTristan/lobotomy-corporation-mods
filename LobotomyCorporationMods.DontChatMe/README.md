# Don't Chat Me! – A Lobotomy Corporation Twitch Integration Mod

> **"Disengagement advised. Interaction inevitable."**

## Overview

**Don't Chat Me!** turns your Twitch chat into an abnormality. Viewers can trigger in-game effects during your Lobotomy
Corporation stream by redeeming interactions—effectively breaching containment in real time.

This mod transforms viewer input into chaos using a websocket bridge between Twitch and a Harmony-patched version of the
game.
